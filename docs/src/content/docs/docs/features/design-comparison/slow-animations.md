---
title: "Slow Animations"
description: "Slow down animations in the iOS Simulator to inspect transitions, debug animation timing, and verify that your UI animations look smooth and correct."
---

Slow animations can be a game changer when optimizing your app’s special animations. By enabling this feature, RocketSim will try to slow down the animation so you can see it frame by frame.
