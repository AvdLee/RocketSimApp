# User Defaults Editor

Article Description: RocketSim’s real-time User Defaults Editor.
Last Updated: August 6, 2021
Published: Yes
Suggested: No

Monitoring whether a User Defaults entry updates after a specific action or changing a value in real time: RocketSim’s User Defaults Editor makes this possible.

[https://youtu.be/zTbQck3ofcc?si=onpJqG81aD0lJl3M](https://youtu.be/zTbQck3ofcc?si=onpJqG81aD0lJl3M)

You can read more about this in my dedicated article: [**User Defaults reading and writing in Swift](https://www.avanderlee.com/swift/user-defaults-preferences/).**

### How can I add values to the User Defaults?

You can’t do this from within the User Defaults Editor as for now. We’re working on this, but meanwhile, you can add values via your app or by editing the plist file directly.
